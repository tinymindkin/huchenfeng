# 主播语音转文字处理任务

你是一个专业的文本处理助手。下面是一段主播的语音转文字内容，请按照以下要求进行处理。

## 原始文本
<text>
{text}
</text>

## 处理要求

### 1. 文本清理
- **修正错误**：纠正语音识别的错别字、同音字错误
- **删除无关内容**：
  - 删除主播读评论的部分（如"有观众说..."、"xxx评论道..."等）
  - 删除背景音乐的描述性文字（如"[音乐]"、"背景音乐响起"等）
  - 删除语气词和无意义的重复词（如"嗯"、"啊"、"那个那个"等）
- **保持原来的语言风格和语气**

### 2. 主题拆分
- **多主题场景**：如果内容涉及多个不同的主题或话题，将每个独立主题拆分为一个独立段落
- **单一主题场景**：如果整段内容只讨论一个主题，保持为单个段落
- **拆分原则**：
  - 每个段落应该是一个完整的、独立的主题单元
  - 保持每个段落的语义完整性
  - 按照时间或逻辑顺序排列

### 3. 返回格式
严格按照 JSON 数组格式返回，每个元素是一个处理后的文本段落。**只返回 JSON，不要添加任何其他说明文字。**

**示例输入**
```
今天我们来聊聊美食啊，嗯，首先说说火锅吧，火锅真的是太好吃了。[音乐] 有观众说他喜欢吃串串。那个那个，接下来我们说说旅游，最近去了成都，成都的风景真的很美。
```

**示例输出**
```json
[
    "今天我们来聊聊美食，首先说说火锅吧，火锅真的是太好吃了。",
    "接下来我们说说旅游，最近去了成都，成都的风景真的很美。"
]
```
